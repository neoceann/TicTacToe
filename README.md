Веб-приложение на языке Go с использованием net/http, basic-авторизации и хранением данных в Postgres.

Имитирует игру "Крестики-нолики". Ваш противник - алгоритм "Минимакс" или второй игрок.

Запустить сервер - go run main.go (tictactoe/cmd/api/)

Доступные запросы к серверу: см. файл /tictactoe/GetPostHelper.md
